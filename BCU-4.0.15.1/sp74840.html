FIXES:
1. The old and current versions are all reserved in Control Panel. This has been
resolved by having the old and current versions located in separate folders.
2. When setting a blank setting value, BCU no longer stops working.
3. Resolved the issue of failing to use parameter/setdefaults to modify BIOS
information when BIOS password is set.
4. When generate repset file with command "BiosConfigUtility64.exe
/getconfig:test.txt /log" BCU no longer stops working.
5. Resolved getting an operation failed error message when doing a setconfig via
BCU.
